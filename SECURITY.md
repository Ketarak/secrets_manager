# Security Policy

## Supported Versions

Only the latest version of Secrets Manager (`passmgr`) is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| >= 1.0  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of Secrets Manager seriously. If you find any security vulnerability, please do NOT open a public issue on GitHub. Instead, report it privately.

To report a vulnerability, please:
1. Open a draft security advisory on GitHub (via the "Security" tab -> "Advisories" -> "New Advisory").
2. Alternatively, email the maintainer directly if an email address is provided.

Please provide a detailed description of the vulnerability, steps to reproduce, and any proof of concept (PoC) if available. We will acknowledge receipt of your vulnerability report within 48 hours and strive to send a fix as soon as possible.
